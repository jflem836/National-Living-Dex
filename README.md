# National Living Dex

Website to keep track of pokemon that you catch between all games. I came up with the idea when I wanted to keep track of all the pokemon I own across different games and see what pokemon I am potentially missing from the games I have. Along with that, I wanted to be able to see what pokemon I am able to catch with the devices I have as well. However, not all of it works perfectly so if you do end up using it, thank you and please give feedback anyway you can!
